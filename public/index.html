Usage

  • Test API:
    GET 140.118.155.33:9999/api/hi
  • See all tiles:
    GET 140.118.155.33:9999/api/gmsdata
  • Get gmsdatas tile by pano_id, x, y:
    GET 140.118.155.33:9999/api/gmsdata/pano_id:id&x:x&y:y
  • Update single tile by id:
    PUT 140.118.155.33:9999/api/gmsdata/:id
  • Delete single tile by pano_id:
    DELETE 140.118.155.33:9999/api/gmsdata/inner/:id
  • Import an tile:
    POST 140.118.155.33:9999/api/gmsdata?pano_id:id&x:x&y:y
  • Count of all tiles:
    GET 140.118.155.33:9999/api/gmsdata/count
  • Delete all tiles:
    DELETE xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  • See all Metadatas:
    GET 140.118.155.33:9999/api/metadata
  • Get metadata item by pano_id
    GET 140.118.155.33:9999/api/gmsdata/pano_id
  • Save a metadata with json:
    POST 140.118.155.33:9999/api/metadata
  • Delete all Metadata:
    DELETE xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

    • =================== Important function below ===================

    You can test by pano_id: W956noQFxUP1cJH0-uYkKA or latLng: 24.9902394, 121.5059227

  • See a panorama by pano_id:
    GET 140.118.155.33:9999/api/panorama/pano_id=pano_id

  • See all XML:
    GET 140.118.155.33:9999/api/panoramaXML
  • Get XML item by pano_id
    GET 140.118.155.33:9999/api/panoramaXML/pano_id=pano_id
  • Get XML item by lat&lng
    GET 140.118.155.33:9999/api/panoramaXML/lat=lat&lng=lng
  • Count of all XML:
    GET 140.118.155.33:9999/api/panoramaXML/count
  • Delete all XML:
    DELETE xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
