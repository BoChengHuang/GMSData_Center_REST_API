Usage

  • Test API:
    GET 140.118.155.33:9999/api/hi
  • See all items:
    GET 140.118.155.33:9999/api/gmsdata
  • Find single item by id:
    GET 140.118.155.33:9999/api/gmsdata/:id
  • Update single item by id:
    PUT 140.118.155.33:9999/api/gmsdata/:id
  • Delete single item by id:
    DELETE 140.118.155.33:9999/api/gmsdata/:id
  • Create an item:
    POST 140.118.155.33:9999/api/gmsdata?name='testName'
